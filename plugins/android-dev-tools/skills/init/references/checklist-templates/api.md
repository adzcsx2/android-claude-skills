## 用户模块 (UserApi)

### 获取用户信息

- **方法**: `GET`
- **路径**: `/user/profile`
- **说明**: 获取当前登录用户信息
- **认证**: 需要

**响应示例**:
```json
{
  "code": 0,
  "data": {
    "userId": "123",
    "username": "user",
    "avatar": "https://..."
  }
}
```

### 用户登录

- **方法**: `POST`
- **路径**: `/user/login`
- **说明**: 用户登录
- **认证**: 不需要

**请求参数**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| username | String | 是 | 用户名 |
| password | String | 是 | 密码 |
